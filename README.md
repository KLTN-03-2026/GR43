# Hệ Thống Ghép Đôi & Phát Hiện Lừa Đảo / Matchmaking & Scam Detection

Build status: backend API pipelines + server health overview.

## Features & Functions

### User Features
- **Registration & Profile**: Users can create accounts and manage their profiles.
- **Matchmaking (Swipe)**: Experience a dynamic matching system for discovery.
- **Real-time Chat**: Message matches instantly via WebSocket.
- **Video/Audio Calls**: Secure WebRTC-powered calling system.
- **Scam Detection**: AI-powered protection flagging suspicious behavior.

### Admin Features
- **User Management**: Monitor and manage user accounts and statuses.
- **Scam Reports**: Review and act on flags generated by the ML model.
- **System Monitoring**: Track real-time server health and call traffic.

## API Endpoints

### Authentication & User
- `POST /api/users`: Register a new user account.
- `POST /api/auth/login`: (Planned) User authentication/login.
- `GET /api/users/{id}`: (Planned) Retrieve user profile details.

### Matchmaking & Real-time
- `GET /api/swipe/recommendations`: (Planned) Get curated match suggestions.
- `POST /api/swipe/action`: (Planned) Like or Dislike a profile.
- `GET /health`: System health check (MongoDB, Redis, API).

## Quick summary
- Frontend: ReactJS + React Native + Tailwind + WebSocket
- Backend: ASP.NET Core 8 + WebSocket server + MongoDB + Redis + JWT
- AI: Python + FastAPI + XGBoost + Sentence Transformers
- Call system: WebRTC + STUN + Coturn, deployed via Docker / Nginx on VPS / Vercel
